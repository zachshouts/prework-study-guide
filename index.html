HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • H1 headings should only appear on a page once.
  • The page's language is set in the opening html tag.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • "." is used to select class names.
  • "#" is used to select ID names.
  • border: top/bottom left/right color: 40px auto gray

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: syncs local branch with remote branch.
  • git push origin branch-name: pushes our local respository to the branch-name.
  • git add -A: stages changes for commit.
  • git commit -m "message": committs all staged changes.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS allows for single or double quotes to encompass a string, but many other languages do not.
  • "var" is function scoped and "let" is block scoped.
  • "let" variables cannot be redeclared and must be declared before use.
